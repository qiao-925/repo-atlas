# Repo Atlas

> GitHub 账号级仓库全局视角与演化分析工作台

Repo Atlas 是一个独立的研究与认知系统，用于分析单个 GitHub 账号下的所有仓库，形成：

- 全局仪表盘
- 仓库地图
- 演化时间线
- 组合分析
- 行动优先级建议
- 面向热门作者账号的模仿学习分析

它不是仓库操作工具。仓库执行与批处理仍然属于 CloneX；Repo Atlas 专注于“看清全局、理解结构、指导行动”。

## 核心能力

- 拉取 GitHub 账号仓库元数据
- 生成全局仪表盘
- 生成仓库聚类与关系图
- 生成演化时间线
- 输出组合分析与优先级建议
- 支持问答式研究入口

## 目标用户

- 管理多个 GitHub 仓库的个人开发者
- 关注账号级项目生态演化的研究者
- 想观察优秀开源作者账号结构的人
